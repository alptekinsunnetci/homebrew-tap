# Homebrew Tap for netplotter

This is a Homebrew tap for installing netplotter.

## Installing netplotter

To install netplotter, run:
```bash
brew tap alptekinsunnetci/tap
brew install netplotter
```

## Computing SHA256

To compute the SHA256 checksum for a file, run:
```bash
shasum -a 256 path/to/your/file
```

Make sure to update the Formula with the correct SHA256 values once computed.
